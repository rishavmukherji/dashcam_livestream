  9. BENEFITS


    • Real-Time Community-Inclusive Police Surveillance
    • Increased Community Reactivity
    • Increased Police Accountability and Transparency → increased community trust
    • Combats Racial Profiling and Excessive Use of Force
    • Evidentiary and Training Purposes
    • Captures Police Perspective

  10. VULNERABILITIES


    1. Access to Live Data by Malicious Actors
    2. Access to Stored Data by Mailicious Actors
    3. Police Tampering
    4. Recording on Private Property
    5. Limited Camera Angle
    6. Unreliable Internet Connectivity
    7. High Startup Cost of Implementation

  11. SOLUTION


    1. Tech implementation:

      • Link live-stream dash-cam to MDT (Mobile Data Terminal)


    2. Policy guidelines:

      • Dash-cams only for select units
      • No recording in response to Code 99 calls
      • Illegal to download police data
      • Write access to footage for select officers
      • Rules for retention of footage
      • No recording on private property
